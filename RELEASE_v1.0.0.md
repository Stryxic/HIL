This release marks the completion and freezing of the HIL Core.

HIL Core v1.0.0 is a deterministic, covariant, noise-aware measurement
instrument for extracting geometric invariants from structured information.

## Included
- Certified timebase
- Stream authority over time
- Covariant Hilbert paths
- Overlay comparability
- Noise-aware resolution
- Informational mass (hard-variant diagnostic)
- CI-enforced invariants
- Presentation kernel reference API

## Excluded (by design)
- Learning
- Optimization
- Prediction
- Semantics
- Decision logic

## Status
Frozen. All future work must be extensions or explicit revisions.
